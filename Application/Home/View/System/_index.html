1001环保系统

我们运用先进的微生物水处理技术，
将收集的自然水源和生活污水直接处理，
达可循环使用或自然排放标准（国家一级出水标准）
是目前全球减少自然保护区环境污染最有效的技术。

-简洁：无需依靠市政集中排污和大型下水管道
-高效：污水即时流入系统，即时被净化并排出
-低成本：制造初始投入成本远低于铺设市政排污
1001 HOTEL还对此系统加以利用，
实现真正的水源净化自循环系统。

处理前污水水质：BOD≈200mg/L

处理后流出水质：BOD≤20mg/L

注：根据《城镇污水处理厂污染物排放标准》中规定，国家一级出水标准为BOD≤20mg/L。

运营系统

  • 精心设计的度假酒店大堂及公共配套功能在野外打造无处不在的野奢度假享受。

  • 创新性地将酒店餐饮、会议、娱乐、后勤等各功能区碎片化设计，大大节约能耗 。

  • 我们有一整套全方位的野外放置蚊虫蛇鼠解决方案，让客人置身于一个免受干扰的户外度假环境。

  • 防潮机制：

    - 智能系统实时检测室内空气湿度与质量，并自动打开设备除湿换气。

    - 安排专人按照户外酒店客房防潮除湿管理准则，定时除湿换气，客人不经意间感受始终如一的室内舒适。

  • 完善的消防装置设计，包括消防通道、消防栓、智能检测、消防喷淋……最大限度预防一切意外的发生。

  • 辅助性设施体现于室内与室外的细微环节，包含安全铃、围栏、照明、标识、智能空气感应……客人想得到的、想不到的，我们都照顾到。
